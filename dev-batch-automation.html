Netpro Trinity ETL Platform

⚙️ 自動化批次 (ETL JOB) 與資料檢核系統

專案目標： 運用 Netpro Trinity 構建跨平台自動化任務排程，針對運動場館營運數據執行多維度 ETL 轉換與實時稽核，確保異質資料庫間的資料嚴謹性與業務邏輯一致性。

🔍 全方位營運數據自動化稽核機制

  • 財務一致性自動對帳： 整合發票系統、支付網關與核心資料庫，實現每日自動對帳流程，精準捕捉金額不符或漏開票等異常，確保帳務零差異。
  • 資源調度邏輯校對： 深度檢核場地預約時段、教練排程與課程配置之重疊性，主動偵測並預警潛在的資源衝突風險，將排程錯誤率降至 0%。
  • 跨系統狀態同步校驗： 針對接手之外包系統進行數據加固，自動偵測訂單從成立至核銷之生命週期狀態，若發現狀態異步即觸發 SMTP 郵件即時告警。
  • 管理決策報表產製： 自動擷取 SQL Server 與 Oracle 異質數據來源，進行清洗與彙整後，定時產出核心營運統計報表供決策層參考。
